"# PyAutoMon" 

Instructions:

Run the script.
Follow the menu to:
1- record
2- save 
3- load
4- replay actions

Press the ESC key to stop recording.
Note:

Ensure you have 
Pip install pynput 
Pip install pyautogui 

(pip install pynput pyautogui).
The recordings folder will store the .macro files.
